# Kali Linux Notes

Kali Linux is one of the operating systems I use for cybersecurity practice and lab environments.

## Common Tools I Practiced

- Nmap
- Wireshark
- Hydra
- Netcat
- Burp Suite
- Aircrack-ng

## Basic Commands

- `sudo su`
- `apt update`
- `apt upgrade`
- `ifconfig`
- `ip a`
- `ping`
- `history`

## Personal Notes

I mainly use Kali Linux for hands-on learning, networking practice, packet analysis, and ethical hacking labs. Some tools are still advanced for me, but I continue practicing and learning step by step.

## Important Reminder

Only use cybersecurity tools in authorized lab environments or systems you have permission to test.
